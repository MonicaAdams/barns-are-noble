A Web Page about Barns

This is a web page about barns. There doesn't seem to be much about the two things in common. One is a building type for storing animals, their food, and equipment. The other is a technological construct as a way to store and disseminate information. But in a way, they are both containers. We used to think about barns as only existing in the rural landscape. We also used to think about web pages as super-modern and high-tech. Although web pages rely on technology to funtion, they can look as weathered and old fashioned as a barn on a country road.
